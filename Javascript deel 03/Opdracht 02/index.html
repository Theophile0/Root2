Waar wordt de event listener gekoppeld aan de slider?

In dit stukje code: sliders[0].addEventListener("change", update); sliders[0].addEventListener("input", update);

Waarom moeten we die op twee soorten events koppelen?

Er moet niet alleen een cijfer aanpassen, je zult ook een string moeten opbouwen met de getallen van de sliders erin. Dit om de CSS te kunnen aanpassen

In de CSS file wordt nergens een rode kleur opgegeven, waar wordt dan wel de rode kleur van het blokje ingesteld?

Deze wordt meegegeven in de .js file met de code: colorDemos[0].style.backgroundColor="red";

Waarom schrijven we telkens sliders[0] en colorDemos[0] en niet gewoon sliders of colorDemos?

Omdat als je .getElementsByClassName gebruikt je een array aanmaakt omdat je meerdere elementen selecteerd en niet 1 element. Door de brackets [0] vraag je het eerste element op.
